# weather-app
A modern Weather Progressive Web App built with HTML, CSS, and JavaScript using the OpenWeather API. Features real-time weather, 5-day forecast, dynamic backgrounds, and weather-based animations (rain, snow, glow effects).

Features:
Search weather by city
Real-time temperature and weather conditions
5-day weather forecast
Dynamic weather icons
Weather-based animations (rain, snow, glow effects)
Dynamic background changes based on weather
Saves last searched city
Installable as a **Progressive Web App (PWA)**
Lightweight & fast (no frameworks)

Technologies Used:
HTML5
CSS3(animations, gradients, glassmorphism)
JavaScript (ES6)
OpenWeather API
Progressive Web App (PWA) technologies  
Web App Manifest  
Service Worker
